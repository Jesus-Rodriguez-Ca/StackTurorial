Set up Ubuntu on Google Cloud

  1. Click here to visit the Google Cloud console page
  2. Create an account
  3. On the bottom right look for the link called Console.
  4. On the top nav bar, select Project
  5. In the nav bar on the left find Compute Engine.
  6. Once you there select VM Instance
  7. Then Create
  8. Name the Instance
  9. Select your Region
  10. Change the Machine type to f1-micro
  11. For Bootdisk click Change > select Ubuntu for OS and Ubuntu 18.04 LTS > Click Select
  12. For Firewall select Allow HTTP Traffic and Allow HTTPS Traffic
  13. Then hit Create and wait for the green circle with white checkmark to indicate your instance is running
  14. Click SSH to the right of the instance that you want to use, keep in mind it will take some time to load
  15. Continue to Node.js to begin installing your stack on this cloud server

Installing Node.js

To install Node.js, paste the following commands then press 'Enter'

sudo apt-get update sudo apt-get install -y curl apt-transport-https ca-certificates && curl --fail -ssL -o setup-nodejs https://deb.nodesource.com/setup_10.x && sudo bash setup-nodejs && sudo apt-get install -y nodejs build-essential

Installing Angular.js

Paste the following command then press 'Enter'

sudo -i npm install -g @angular/cli

Type “y” or “n” and press 'Enter' when asked to share data with Google (makes no difference for this tutorial)
